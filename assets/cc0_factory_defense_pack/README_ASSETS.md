# CC0 Factory Defense Asset Pack

Original free placeholder art for a Godot 4 top-down automation and tower-defense prototype.

## Contents

- 25 tile sprites for ground, ore, rock, enemy paths, and spawn markers.
- Animated block sprite sheets for conveyors, drills, turret firing, and the 3x3 core.
- Static blocks for walls, a router placeholder, and a power-node placeholder.
- Item sprites, projectiles, enemy walk loops, hit effects, and toolbar icons.

Sprite sheets are horizontal. See `animation_metadata.json` for frame sizes and frame counts.
